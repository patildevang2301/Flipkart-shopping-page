# Flipkart Homepage Clone

## Project Overview

This project is a front-end clone of the Flipkart homepage created using **HTML** and **CSS**. It recreates the look and feel of the Flipkart website, including the navigation bar, category section, promotional banners, product listings, featured deals, and footer section.

The purpose of this project is to practice web development fundamentals, layout design, and styling techniques using pure HTML and CSS.
